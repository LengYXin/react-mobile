# SchoolBusWeb

## 安装依赖 以及启动命令
```
        /nativeWeb/package.json
        npm start                    启动 web
        npm run build                打包程序

        android真机预览效果 (需要android adb环境)
         adb reverse tcp:端口 tcp:端口
        android浏览器打开
         localhost:端口

```
